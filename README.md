# Встречайте - OtakuBoard
**OtakuBoard** — веб-приложение для коллекционирования цифрового контента в виде карточек. Работайте прямо в браузере, а все данные хранятся локально на вашем устройстве.

Приложение даёт возможность создавать и настраивать карточки под свой вкус: указывайте тематики (например: аниме, персонажи, музыка, фильмы, книги и другое) и храните в них любой контент. Карточки можно фильтровать, объединять в группы и составлять личные топы.

Посмотрели аниме или хороший фильм? Сохраните его в виде небольшого достижения и создайте собственную коллекцию цифрового контента в приложении OtakuBoard!

Приложение доступно для ОС Windows и Linux.

# Технологии и инструменты
- Языки программирования: Python, JavaScript
  
- Интерфейс: Веб-интерфейс
  
- Связка Frontend-Backend: Eel (Python <-> JavaScript)
  
- Формат хранения данных: JSON
  
- Платформа: Desktop (Windows / Linux)
