# Что такое OtakuBoard
- **OtakuBoard** - это Open Source, Web-приложение для коллекционирования различного цифрового контента в виде "контент-блоков" (Визуальное представление контента в приложении) на сайте и файлах локально на ПК. Приложение даёт возможность создать группу (Например: Аниме, персонажи, музыка, фильмы, книги и другое) и хранить в ней определенный контент, фильтровать его, создавать личные топы и рейтинги контента. В самих "контент-блоках" возможно указать название контента, год, автора, описание, ссылки, жанры, оценку, место в личном топе, теги и группу. Приложение доступно для  Windows/Linux ОС.
