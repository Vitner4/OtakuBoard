# What is OtakuBoard
- **OtakuBoard** is an Open Source, Web application for collecting various digital content in the form of "content blocks" (Visual representation of content in the application) on the site and JSON files locally on the PC. The application allows you to create a group (For example: Anime, characters, music, movies, books, etc.) and store certain content in it, filter it, create personal tops and content ratings. In the "content blocks" themselves, you can specify the name of the content, year, author, description, links, genres, rating, place in the personal top, tags and group, etc.

- Have you watched anime or some good movie? Then save it as a small achievement and create your own collection of digital content in the OtakuBoard application!
# Goal
- Create a simple and convenient desktop Web application for storing digital content in files locally on the user's device with Windows / Linux OS.
